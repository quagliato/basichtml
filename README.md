# basichtml

basichtml is a simple file structure to help your productivity on writing plain
html websites.

## What comes imported
* jquery-2.1.3
* jquery-ui-1.11.2
* jquery.easing-1.3
* jquery.mask
* font-awessome-4.3.0

## What functions are built in
* A fully socialmedia-ready <head>, including open graph and google+ tags;
* Full reset css;
